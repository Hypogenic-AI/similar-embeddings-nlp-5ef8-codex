# Cloned Repositories

## MUSE
- URL: https://github.com/facebookresearch/MUSE
- Purpose: Unsupervised and supervised cross-lingual word embedding alignment
- Location: code/muse/
- Key files: `supervised.py`, `unsupervised.py`, `evaluate.py`, `data/get_evaluation.sh`
- Notes: Provides bilingual dictionaries and evaluation datasets; supports Procrustes alignment and adversarial mapping.

## VecMap
- URL: https://github.com/artetxem/vecmap
- Purpose: Cross-lingual word embedding mapping with supervised/semi/unsupervised modes
- Location: code/vecmap/
- Key files: `map_embeddings.py`, `eval_translation.py`, `eval_similarity.py`, `get_data.sh`
- Notes: Strong baseline for dictionary induction and similarity evaluation.

## XLM
- URL: https://github.com/facebookresearch/XLM
- Purpose: Cross-lingual language model pretraining (XLM) and related models (XLM-R)
- Location: code/xlm/
- Key files: `train.py`, `preprocess.py`, `get-data-wiki.sh`, `generate-embeddings.ipynb`
- Notes: Provides contextual multilingual representations and embedding extraction utilities.
